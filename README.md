### [Rush Search](https://github.com/brivera2005/RushSearch)

Ultra-fast **Windows file search**. Press **Ctrl+Space** anywhere for a floating overlay — fuzzy matching, full-drive indexing, shell actions, and system-tray background mode.

`Electron` · `JavaScript` · Windows 10/11

[![Rush Search repo](https://img.shields.io/badge/Rush_Search-View_repo-0D1117?style=for-the-badge&logo=windows&logoColor=2DD4BF&labelColor=14B8A6)](https://github.com/brivera2005/RushSearch)

### [RushTV](https://github.com/brivera2005/RushTV)

Local **Windows IPTV player** with a dark RushTV theme. Connect via **Xtream Codes** or paste an **M3U playlist** — playback runs through **VLC** embedded in the app.

`Python` · `CustomTkinter` · `VLC` · Windows 10/11

### ⭐ [TomeWizard](https://www.tomewizard.com) — *featured*

**Five picks. Zero doomscrolling.**

TomeWizard helps people discover books, movies, shows, music, and games based on **how they feel now** and **how they want to feel after** — then saves picks to a personal library with real covers.

It also offers **developmental editing** for writers: structural reports, chapter pacing maps, character arc audits, and revision priorities — the layer that makes readers stay up past midnight (not line edits).

`Next.js` · mood-matched curation · writer tooling
