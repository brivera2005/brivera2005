### [Strewn](https://github.com/brivera2005/strewn-v5-overhaul) — *the game*

**Share the weight. Stack the loot. Deploy the minions.**

A top-down **burden-strategy roguelite** about pain mitigation in a shattered world. You're a Painweaver: lash enemies, route colored suffering through sink nodes, vent when the Burden Bar threatens overflow, and survive escalating 20-minute runs. CYOA tutorial, family mechanics, minion deployment, loot loops, and a triage command center.

Portable Windows `.exe` — no terminal, no browser tab. Download and play.

`React` · `TypeScript` · `Vite` · `Electron` · `Tauri`
